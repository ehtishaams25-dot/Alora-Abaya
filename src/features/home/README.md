# Home Feature (`src/features/home/`)

## Overview
This feature manages the homepage components and boutique luxury presentation for **Layali (ليالي)**. It adheres to the **Luxury Minimal** design direction with generous white space, refined typography, and smooth micro-animations.

## Components
- `HeroSection.tsx`: The primary landing hero showcase featuring an asymmetrical split with editorial Playfair Display typography, luxury CTA buttons (`.btn-primary`, `.btn-secondary`), trust indicators, and a cinematic looping video showcase with floating craftsmanship badges.

## Public API (`index.ts`)
- `HeroSection`
