# Template Feature

The template feature demonstrates the project's expected feature boundary.

## Responsibilities

- Show the landing overview for this starter template.
- Demonstrate the components, hooks, services, and types layering pattern.
- Export only the public API through `index.ts`.

## Public API

- `TemplateOverview`
- `useTemplateOverview`
- `TemplatePrinciple`
- `TemplateStat`
